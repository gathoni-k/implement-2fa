## Implement 2 Factor Authentication using Authy and Node

### Project set up
* Clone repo
```bash
git clone git@github.com:gathoni-k/implement-2fa.git
```
* In a .env file:
```JavaScript
AUTHY_API_KEY='5NVmSSIEQ*****gfumvobcMUTQ9*****' // get Api key from the Authy dashboard
MONGO_URI='mongodb://127.0.0.1:27017/2fa'
PORT='3000'
```
* Install dependencies
```bash
npm install
```
* Run app
```bash
npm start
```
